# Sistem Manajemen Les & SPP

Aplikasi manajemen les privat dan pembayaran SPP (Sumbangan Pembinaan Pendidikan) berbasis web.

## Fitur

### 🔐 Sistem Login
- **Admin**: username `admin`, password `1234`
- **User**: username `user`, password `1234`

### 👨‍🎓 Data Siswa
- Tambah data siswa baru
- Lihat daftar siswa
- Hapus data siswa
- Status aktif/cuti

### 📋 Absensi
- Absensi masuk & pulangsekaligus
- Simpan absensi harian
- **Export ke Excel** dengan format rapih
- Riwayat absensi tersimpan

### 💳 Pembayaran SPP
- Tambah pembayaran SPP
- Filter berdasarkan bulan
- Status lunas/belum lunas
- **Export ke Excel** dengan format rapih

### 📊 Export Excel
- Export data absensi ke Excel
- Export data pembayaran ke Excel
- Format otomatis terstruktur rapih
- Timestamp pada nama file

## Teknologi

- **HTML5** - Struktur halaman
- **CSS3** - Styling modern dengan Font Poppins
- **JavaScript** - Logika aplikasi
- **SheetJS (xlsx)** - Export ke Excel
- **LocalStorage** - Penyimpanan data lokal

## Cara Menjalankan

1. Buka file `index.html` di browser
2. Login dengan kredensial yang tersedia
3. Gunakan fitur sesuai kebutuhan
